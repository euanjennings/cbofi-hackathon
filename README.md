# Bank of Ireland Data Challenge — NayaOne Hackathon

Machine learning project completed as part of the **Bank of Ireland Data Challenge**, hosted on the [NayaOne](https://www.nayaone.com/) collaborative sandbox platform.

## Overview

The challenge involved analysing financial data to build a **fraud detection classifier**, supported by dimensionality reduction and exploratory analysis techniques.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `BoI_Data_Challenge.ipynb` | Main challenge workbook — problem statement, EDA, and modelling |
| `main.ipynb` | Full model pipeline |
| `main_eval.ipynb` | Model evaluation and performance metrics |
| `PCA.ipynb` | Principal Component Analysis for dimensionality reduction |
| `LCA.ipynb` | Latent Class Analysis |
| `Data_Challenge_Getting_Started_Guide.ipynb` | NayaOne sandbox getting started guide |

## Key Files

- `fraud_classifier.pkl` — Trained fraud detection model
- `correlation_matrix.png` — Feature correlation heatmap
- `Model_Results/` — Saved model outputs and results

## Tech Stack

- Python (Jupyter Notebooks)
- scikit-learn
- pandas / numpy
- matplotlib / seaborn
- NLTK

## Platform

Developed in the [NayaOne](https://www.nayaone.com/) collaborative JupyterLab sandbox — a real-time multi-user data environment provided for the hackathon.
